# AI-Search-Engine-Project
An AI-powered Search Engine built using Python, Streamlit, and Google's Gemini API. The application provides intelligent, context-aware responses to user queries through a simple and interactive web interface, showcasing the integration of generative AI with modern web applications.
AI Search Engine using Gemini API

📌 Overview

This project is an AI-powered Search Engine built using Python and Streamlit. It integrates Google's Gemini API to generate intelligent responses based on user queries.

✨ Features

- AI-powered search using Google Gemini
- Interactive and simple Streamlit interface
- Fast and accurate responses
- Easy to set up and use

🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API

📂 Project Structure

AI-Search-Engine/
├── app.py
├── requirements.txt
├── README.md
└── .gitignore

🚀 Installation

1. Clone the repository.
2. Install the required packages:
   pip install -r requirements.txt
3. Add your own Gemini API key.
4. Run the application:
   streamlit run app.py

🔑 API Key

This project does not include a Gemini API key. Create your own API key from Google AI Studio and configure it before running the application.
